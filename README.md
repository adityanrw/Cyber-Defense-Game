# Cyber Defense Game

Cyber Defense Game adalah project game Unity bertema pertahanan server dari serangan cyber.

## Dibuat Dengan
- Unity
- C#

## Isi Repository
- Assets
- Packages
- ProjectSettings

## Cara Membuka Project
1. Clone atau download repository ini.
2. Buka Unity Hub.
3. Pilih Add project from disk.
4. Arahkan ke folder project ini.
5. Buka menggunakan versi Unity yang sesuai.
